## ✨ Features

*   **Clean & Modern UI:** An intuitive and visually pleasing interface with a carefully curated color palette and sophisticated font.
*   **Engaging Animations:** Fluid animations powered by Framer Motion for an exciting user experience.
*   **Fully Responsive:** Mobile-first design that looks and functions flawlessly on all screen sizes.
*   **Animated Score Counter:** Smoothly animated score updates for immediate feedback.
*   **Light & Dark Modes:** A theme toggle for a personalized viewing experience.
*   **High-Quality Graphics:** Crisp and scalable vector icons for high-resolution displays.

## 🛠️ Technologies Used

*   **React:** A JavaScript library for building user interfaces.
-   **Vite:** A fast build tool and development server for modern web projects.
*   **Framer Motion:** A production-ready motion library for React.
*   **React Icons:** A library for including popular icons in your React projects.

## 💻 Setup Locally

To get the project up and running on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/udaykrizzz19/Rock-Paper-Scissors-Game.git
    cd Rock-Paper-Scissors-Game
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```
    Open your browser and navigate to `http://localhost:5173` to see the app in action.
